El CSS es un lenguaje de diseño que se utiliza para dar estilo a los documentos HTML. Fue creado por Håkon Wium Lie y Bert Bos en 1996. El CSS se utiliza para definir el diseño de los documentos HTML, incluyendo el diseño, el color, la fuente y otros aspectos de la presentación de un documento. El CSS se utiliza para separar el contenido de la presentación de un documento, lo que permite a los diseñadores y desarrolladores de sitios web cambiar el diseño de un sitio web sin tener que cambiar el contenido del sitio web. Destacados

El CSS se ha desarrollado a lo largo de los años para incluir nuevas características y funcionalidades. En 1998, el CSS2 fue lanzado, que incluía nuevas características como la capacidad de definir estilos para la impresión y la capacidad de definir estilos para diferentes medios, como la pantalla y la impresión. En 2011, el CSS3 fue lanzado, que incluía nuevas características como la capacidad de definir estilos para diferentes dispositivos, como los teléfonos móviles y las tabletas. En 2018, el CSS4 fue lanzado, que incluía nuevas características como la capacidad de definir estilos para diferentes resoluciones de pantalla y la capacidad de definir estilos para diferentes idiomas.
